The DOM API does not provide a function to remove all children of an element. A straightforward way to do this is to remove each child element: {% include code-snippet.html file="remove-children-js/Naive.js" %}

This works, but there is a faster alternative: select all children using the DOM Range API and delete the contents of the range. This is supported in all modern browsers (in Internet Explorer since version 9). {% include code-snippet.html file="remove-children-js/Range.js" %}
